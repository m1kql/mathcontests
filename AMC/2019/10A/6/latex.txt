For how many of the following types of quadrilaterals does there exist a point in the plane of the quadrilateral that is equidistant from all four vertices of the quadrilateral?
 
 \begin{itemize} \item a square \item a rectangle that is not a square \item a rhombus that is not a square \item a parallelogram that is not a rectangle or a rhombus \item an isosceles trapezoid that is not a parallelogram \end{itemize}$\textbf{(A) } 0 \qquad\textbf{(B) } 2 \qquad\textbf{(C) } 3 \qquad\textbf{(D) } 4 \qquad\textbf{(E) } 5$
