 \begin{center} \[x_m\leq 4+\frac{1}{2^{20}}.\] \end{center} In which of the following intervals does $m$ lie?
 
$\textbf{(A) } [9,26] \qquad\textbf{(B) } [27,80] \qquad\textbf{(C) } [81,242]\qquad\textbf{(D) } [243,728] \qquad\textbf{(E) } [729,\infty)$
