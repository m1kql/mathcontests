For any three real numbers $a$, $b$, and $c$, with $b\neq c$, the operation $\otimes$ is defined by:
 
 \begin{center} \[\otimes(a,b,c)=\frac{a}{b-c}\] \end{center} 
 
What is $\otimes(\otimes(1,2,3),\otimes(2,3,1),\otimes(3,1,2))$?
 
$\mathrm{(A) \ } -\frac{1}{2}\qquad \mathrm{(B) \ } -\frac{1}{4} \qquad \mathrm{(C) \ } 0 \qquad \mathrm{(D) \ } \frac{1}{4} \qquad \mathrm{(E) \ } \frac{1}{2}$
